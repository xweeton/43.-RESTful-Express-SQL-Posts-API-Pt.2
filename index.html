Welcome to the API!

How to use this API:

  • POST /posts : Create a new post. The body should be a JSON object with 'title', 'content', and 'author' keys.

  • GET /posts : Get all posts.

  • GET /posts/:id : Get a post by ID. Replace ':id' with the ID of the post you want to retrieve.

  • GET /posts/author/:authorName : Get all posts by a specific author. Replace ':authorName' with the author's name.

  • GET /posts/dates/:startDate/:endDate : Get all posts created within a date range. Replace ':startDate' and ':endDate' with the desired date range in the format 'YYYY-MM-DD'.

  • PUT /posts/:id : Update an existing post. Replace ':id' with the ID of the post you want to update. The body should be a JSON object with the 'title', 'content', 'author' keys you want to update.

  • DELETE /posts/:id : Delete a post. Replace ':id' with the ID of the post you want to delete.

  • DELETE /posts/author/:authorName : Delete all posts by a specific author. Replace ':authorName' with the author's name to delete all their posts.
